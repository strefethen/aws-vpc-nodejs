# aws-vpc-nodejs
This NodeJS application creates a Virtual Private Cloud on AWS.

## Prerequisites
* AWS Account
* Setup [aws-sdk](https://aws.amazon.com/sdk-for-node-js/) node package from Amazon

## Run
    $ git clone https://github.com/strefethen/aws-vpc-nodejs.git
    $ cd aws-vpc-nodejs
    $ npm install
    $ tsc --project .
    $ node built/index.js
